# Fiji Plugin Requirements for Nuclei StarDist Detection
#
# Install via: Help → Update… → Manage update sites… → tick the sites below,
# then Close → Apply changes → restart Fiji.

# Required plugins / update sites
Bio-Formats    # bundled with Fiji; opens .lif / .nd2 / .tif / .ome.tif
StarDist       # nuclei segmentation (update site: StarDist)
CSBDeep        # StarDist dependency – model execution (update site: CSBDeep)
TensorFlow     # runtime backend for CSBDeep/StarDist (update site: TensorFlow)
IJPB-plugins   # morphology / label operations (update site: IJPB-plugins)
MorphoLibJ     # morphological filters and label maps (update site: IJPB-plugins)

# Optional
TrackMate      # nuclei tracking across frames/series (update site: TrackMate)
